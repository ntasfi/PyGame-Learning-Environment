# DonkeyKong-Pygame
Author: Batchu Vishal

A spinoff of the original Donkey Kong game, which is simpler than the original. The codebase is written in python (pygame).

Instructions:
1)Use wasd keys to move around and climb ladders.
2)Use space to jump.
